## Why this exists

Agent pipelines fail in messy ways. Teams need a common way to inspect prompts, tool calls, and model outputs without digging through logs.

## Current scope

- Session-level traces with timeline drill-down
- Evaluation snapshots for regression checks
- Latency and token cost dashboards by workflow step

## Next milestone

Automated alerting for quality regressions in high-traffic flows.
